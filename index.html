Notes for Graders:

  • Use the mouse to control the camera!
  • Use the W, A, S, D keys to move around
  • Use the Q, E keys to rotate left and right respectively
  • Use the F key to place blocks in a stack in front of you
  • Use the R key to remove a block stack in front of you
  • Use the Shift key to fly up
  • Use the Ctrl key to fly down
